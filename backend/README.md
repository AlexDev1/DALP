# DALP backend 
## Installation

`npm install`

## Setting up

- Create a config folder in backend/
- Create dev.json or test.json file based on your usecase
- Add the following code to dev/test.json

`{"dbURI":"YOUR DATABASE URI"}`

## Running the main server

`npm start`

## Running the socket server

`npm run socket`

## Testing

`npm test`
